Developed a full-stack food community platform using Next.js, SQLite, CSS, and JavaScript. Implemented features such as Browse Meals and Foodie Community, enabling users to upload and share meal images with descriptions. Built custom API routes for server-side data handling and optimized performance through SSR (Server-Side Rendering) and SSG (Static Site Generation).
